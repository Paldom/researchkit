# Security Policy

## Supported Versions

Only the latest release of researchkit receives security fixes.

## Reporting a Vulnerability

Please do **not** open a public issue for security problems.

Report vulnerabilities privately via GitHub's private vulnerability reporting:
<https://github.com/Paldom/researchkit/security/advisories/new>

You can expect an acknowledgement within 7 days. Please include steps to
reproduce and the affected version. Fixes will be released as soon as
practical and credited unless you prefer otherwise.
